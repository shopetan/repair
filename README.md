# Asobeat
Repair 遠隔ペアプログラミングシステム


Repair で 空間を Hackする．

# Description
Repairは，遠隔地でのペアプログラミングを支援するWebアプリケーションです．
socket.ioによる非同期通信と，APIサーバによって動的なテキスト編集，ドキュメント保存を可能にします．
